#  Math Object / Lecture 10 Code
### JS
```javascript
const num1 = 100;
const num2 = 50;
let val;

// Simple math with numbers
val = num1 + num2
val = num1 * num2 // *= multiply
val = num2 - num1
val = num1 / num2
val = num1 % num2 // %= remainder

// Math Object
val = Math.PI;
val = Math.E;
val = Math.round(2.4);
val = Math.ceil(2.3); //rounds up
val = Math.floor(2.9);
val = Math.sqrt(64);
val = Math.abs(-3.57); //gives you positive of number
val = Math.pow(8, 2);
val = Math.min(7, 6, 19, 17, 31, 5)
val = Math.max(7, 6, 19, 17, 31, 5)
val = Math.random();
```
